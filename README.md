# Cidesoft Web

Official web site for Cidesoft.

## Steps to Run and Build

### Running the Project

1. **Install Dependencies**:

```sh
npm install
```

2. **Start Development Server**:

```sh
npm run dev
```

### Building the Project

1. **Build for Production**:

```sh
npm run build
```

2. **Preview the Build**:

```sh
npm run preview
```
